# Linear Attention Vision Transformers for Particle Physics
**End-to-End Mass Regression and Classification at the CMS Detector**

*This project was developed as part of the ML4SCI / CMS End-to-End Deep Learning research initiative to benchmark $O(N)$ linear attention mechanisms against standard quadratic Vision Transformers for dense particle collision data.*

**Applicant:** Ray Arcand (ray.arcand@utexas.edu)  
**GSoC 2026 Target:** Specific Task 2h (CMS/E2E Deep Learning)

## Project Overview
Standard Vision Transformers (ViTs) suffer from quadratic computational and memory complexity with respect to image sequence length. In high-energy physics, particle collision images generated by the Large Hadron Collider (LHC) are incredibly dense, making standard MHSA (Multi-Head Self Attention) a bottleneck for inference latency and VRAM.

This repository implements a **Dual-Head Linear Attention Vision Transformer** in PyTorch. By utilizing linear-scale attention approximations, this architecture efficiently processes `.h5` particle hit matrices to simultaneously:
1. **Classify** the generated heavy particle type.
2. **Regress** the mass of the particle.

## Repository Structure
* `notebooks/`: Contains the end-to-end Jupyter Notebook detailing data ingestion, model architecture, pre-training, fine-tuning, and benchmarking.
* `weights/`: Stores the `.pth` files for the pre-trained, fine-tuned, and baseline models.
